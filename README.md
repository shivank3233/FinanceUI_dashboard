# CoinFlow Financial Dashboard

CoinFlow is a gamified, beautiful, and interactive React dashboard designed to help you track your monthly expenses, view financial insights, and achieve your goals with fun gamification elements.

## Features
- **Dynamic Dashboard Overview:** Track total balance, income, expenses, and a focused 30-day view of your spending.
- **Goal Checking & Achievements:** Experience gamified streaks, badges, and progress milestones as you hit predefined targets.
- **3D Tilt Elements & Aesthetics:** Beautiful glassmorphism UI paired with smooth interactive hover cards.
- **Secure Role Based UI**: Separate views for Viewers and Administrators.

## Administrator Mode
By default, the dashboard may be viewed in a read-only `Viewer` mode or `Administrator` mode depending on the active state. 
To switch into the `Administrator` mode and edit the dashboard state, select "Administrator (Edit)" in the sidebar.

> **Default Password:** `admin123`

## Running Locally
Make sure you have Node.js installed. In the project directory, run:
```bash
npm install
npm run dev
```
